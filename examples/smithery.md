# Use the Swfte MCP server via Smithery

[Smithery](https://smithery.ai) is a marketplace for MCP servers with one-click install. The Swfte server is published as `@swfte/mcp-server` and can be found at:

> https://smithery.ai/server/@swfte/mcp-server

To install:

```bash
npx -y @smithery/cli install @swfte/mcp-server --client claude
```

…and supply your `SWFTE_API_KEY` and `SWFTE_WORKSPACE_ID` when prompted.

Get an API key: [swfte.com/settings/api-keys](https://www.swfte.com/settings/api-keys).
About Swfte: [swfte.com](https://www.swfte.com).
